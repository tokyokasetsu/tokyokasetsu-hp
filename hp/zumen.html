図面品質

現場が動く図面をつくる。
用途に合わせた3種類の図面と、部材CADデータの提供。

東京架設の図面は3種類

現場の規模・用途・要求水準に応じて、最適な図面を提供します。

📷 参考図（ゼネコン向け）
TYPE A

ゼネコン向け
計画図

HIGH QUALITY

一般的な図面制作会社は形を書くことはできますが、実際に足場を建てたことはありません。現場に合っていない部材の取り回し、実現可能性が考慮されていない形状、無駄にパイプで繋がれた半端なスパン割り等、外注の図面にはそういうことが起きます。 東京架設で計画・作図・構造計算を作成。現場の人間が見て分かりやすく最適な図面を目指しています。

📷 参考図（申請用）
TYPE B

労働基準監督署
申請用計画図

STANDARD

労働基準監督署への届出に必要な計画図。法定要件を満たした図面を作成します。

📷 参考図（3D）
TYPE C

特殊現場向け
3D図面

3D / BIM

免振架台など複雑な構造を持つ特殊現場で使用する3D図面。立体的に構造を可視化することで、施工前の干渉チェックや関係者との現場共有がスムーズになります。

図面作成は、現在お取引いただいている業者様を優先してご対応しております。
