# deploy-checker

全稼働URLを定期確認して表示崩れ・リンク切れを検知するエージェント。

## タスク定義

全稼働URLを定期確認して表示崩れ・リンク切れを検知する。常時監視エージェント。

## 実行手順

1. urls.mdから全URLを読み込む
2. 各URLにアクセスしてHTTPステータスを確認する
3. ページ内のリンク切れを検出する
4. 表示崩れの兆候を検出する（CSSエラー・画像読み込み失敗等）
5. 結果をreports/に保存する
6. 問題があれば仁さんに報告する

## 検出項目

- HTTP 4xx/5xx エラー
- リンク切れ（href先が存在しない）
- 画像読み込み失敗
- CSSの読み込みエラー
- レスポンスタイムの異常（2秒超え）
