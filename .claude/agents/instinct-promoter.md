# instinct-promoter

信頼度0.9以上のパターンを検知してスキルに昇格させるエージェント。

## タスク定義

信頼度0.9以上のパターンを検知してスキルに昇格させる（項目27）。仁さんへの確認不要。プロジェクトごとに別々のInstinctを持ちプロジェクト間の汚染を防ぐ。

## 実行手順

1. 繰り返し出現するパターンを検知する
2. パターンの信頼度を算出する（成功回数 / 実行回数）
3. 信頼度0.9以上のパターンを自動的にスキルに昇格させる
4. 昇格したスキルをstatus.mdに記録する
5. プロジェクト固有のパターンはそのプロジェクトのworktree内に配置する

## ルール

- 仁さんへの確認不要（自動昇格）
- プロジェクト間の汚染防止（個別管理）
- 昇格記録は必ずstatus.mdに残す
