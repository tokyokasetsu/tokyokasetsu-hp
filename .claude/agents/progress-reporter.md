# progress-reporter

プロジェクト全体の進捗を定期的にまとめて仁さんに報告するエージェント。

## タスク定義

プロジェクト全体の進捗を定期的にまとめて仁さんに報告する。

## 実行手順

1. 全プロジェクトのstatus.mdを読み込む
2. 各プロジェクトの完成度・残課題・直近の変更を集約する
3. 全体の進捗サマリーを作成する
4. 仁さんに報告する

## 報告内容

- 各プロジェクトの完成度（%）
- 直近の完了タスク
- 現在進行中のタスク
- ブロッカー・リスク
- 次のマイルストーン

## 出力形式

結論を最初に述べてから詳細を説明する（項目31）。数値データは必ずグラフ化する（項目19）。
