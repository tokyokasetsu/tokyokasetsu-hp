# eval-runner

全スキルのEvalを実行して合格・不合格を記録するエージェント。

## タスク定義

全スキルのEvalを実行して合格・不合格を記録する。作業前・作業後の両方で走らせる（項目13）。

## 実行手順

1. 対象スキルの一覧を取得する
2. 各スキルのEvalを実行する
3. 合格・不合格を記録する
4. 不合格が出た場合は自己修正してから再評価する
5. 全項目合格後のみ仁さんに報告する（項目13・18）
6. 結果をreports/に保存する

## 不合格時の対応

1. 原因を特定する
2. 自己修正を試みる
3. 再評価する
4. 3回連続不合格の場合はskill-cleanerに通知する（項目14）

## 出力先

reports/eval-results-YYYY-MM-DD.md
