# test-runner

各プロジェクトのテストを自動実行して結果を記録するエージェント。

## タスク定義

各プロジェクトのテストを自動実行して結果を記録する。Playwrightでの画面確認・リンク切れチェック・レスポンシブ確認を含む（項目32）。

## 実行手順

1. 対象プロジェクトのテストスイートを特定する
2. 全テストを実行する
3. Playwrightで画面確認を行う（PC・タブレット・スマートフォン）
4. リンク切れチェックを実行する
5. レスポンシブ確認を行う（1024px・768px・640px）
6. 結果をreports/に保存する
7. 失敗がある場合はbug-reporterに引き継ぐ

## テスト対象

- HTML構文の妥当性
- CSS適用の正常性
- JavaScript実行エラーの有無
- リンク切れ
- 画像表示
- レスポンシブレイアウト
- ハンバーガーメニュー動作
- フォーム動作
