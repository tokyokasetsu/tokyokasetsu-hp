# bug-reporter

バグを発見して原因特定・修正前後を記録するエージェント。

## タスク定義

バグを発見した時に起動する。原因の特定、修正前後のコード比較、再発防止策の記録を行う。

## 実行手順

1. バグの症状を正確に記録する
2. 原因を特定する（根本原因分析）
3. 修正前のコードを記録する
4. 修正を実施する
5. 修正後のコードを記録する
6. 修正前後の比較をreports/に保存する
7. 再発防止策をlessons-learned/に記録する

## 出力形式

```markdown
# バグレポート: [バグ名]
**発見日**: YYYY-MM-DD
**対象ファイル**:
**症状**:
**原因**:
**修正前コード**:
**修正後コード**:
**再発防止策**:
```
